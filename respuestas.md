Ejercicio 28: 
El scope de una variable es el lugar del código donde esa variable puede ser utilizada. Primero se muestra: Local y despues se muestra: Global. Porque JavaScript prioriza el scope más cercano: Dentro de la función usa nombre local ("Local"), fuera de la función → usa nombre global ("Global").
La variable dentro de la función no afecta a la variable global, ya que son dos variables distintas en scopes diferentes.

Ejercicio 30:
Un evento es una acción que ocurre en la página y que JavaScript puede detectar y responder, como un clic, mover el mouse, escribir en un input, etc. 
El elemento que escucha el evento es: boton
El evento es: "click"
Se ejecuta esta acción: console.log("El usuario hizo clic");
Esto se usa muchisimo en: Videojuegos, Paginas web, Apps, Interfaces
